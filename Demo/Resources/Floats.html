Floated Images

An image with float:left sits at the left content edge and the text of the paragraph wraps around it on the right side. Once the text reaches the bottom edge of the image it returns to the full width of the column, exactly like in a web browser. Floated Blocks

A div with float:right and a fixed width becomes a column that the main text flows around.

Block elements can float too: the floated box keeps its own width — either the CSS width or, without one, the natural width of its content — and the surrounding paragraphs wrap along its edge.
